Internet :

Global system of interconnected computer networks that uses the Internet protocol suite (TCP/IP)to communicate between networks and devices.The internet was born in 1983 as a military tool by the goverment of the USA.

Protocol (concept, types, RFC)

Is the network layer communications protocol in the Internet protocol suite for relaying datagrams across network boundaries. Its routing function enables internetworking, and essentially establishes the Internet.

RFC "Request for Comments":

Is a publication in a series from the principal technical development and standards-setting bodies for the Internet, most prominently the Internet Engineering Task Force (IETF), the protocols detailed are HTTP and FTP.
ISO-OSI levels

Is a conceptual model that 'provides a common basis for the coordination of [ISO] standards development for the purpose of systems interconnection'.In the OSI reference model, the communications between a computing system are split into seven different abstraction layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.
HTTP protocol

Is an application layer protocol in the Internet protocol suite model for distributed, collaborative, hypermedia information systems.HTTP is the foundation of data communication for the World Wide Web, where hypertext documents include hyperlinks to other resources that the user can easily access, for example by a mouse click or by tapping the screen in a web browser.


  • concept, utility, characteristics : functions as a request–response protocol in the client–server model,it delivers content thru client-server communication.

  • structure of an HTTP request and response packet : Browser sends a request sent to the web server that hosts the website. The web server then returns a response as a HTML page or any other document format to the browser. Browser displays the response from the server to the user.

  • HTTP methods and their utility : This methods are GET , HEAD , POST , PUT , DELETE , CONNECT , OPTIONS , TRACE this methods do explained in the order shown : retrieve info , same as GET but only gets the header , send data , replace thru uploaded content , remove target given , establish a connection , describe communication and perform a message loop.
The process of loading a web page

A page load begins when a user selects a hyperlink, submits a form, or types a URL in a browser. This is also referred to as the initial request or the navigation start. The user's action sends a request across the network to the web application server. The request reaches the application for processing.
TCP (concept, utility, well-known ports, structure of TCP segments, sockets)

Is one of the main protocols of the Internet protocol suite. It originated in the initial network implementation in which it complemented the Internet Protocol (IP).

TCP is connection-oriented, and a connection between client and server is established before data can be sent. The server must be listening (passive open) for connection requests from clients before a connection is established.


Well-known ports:
port numbers 0 to 1023 are reserved for privileged services and designated as well-known ports.
Example port 80 is the HTTP port to acces the internet


Segmentation:
is the process of carving up information into smaller pieces. The documentation for Transmission Control Protocol (TCP) refrers to what it calls 'data streams'. By segmenting the data, TCP creates chunks of data that can be routed separately over whatever connections are needed in order to reach the destination.


Sockets:
A socket is one endpoint of a two-way communication link between two programs running on the network.

IP (concept, utility, IP addresses, routing, structure of IP datagrams)

IP stands for "Internet Protocol," which is the set of rules governing the format of data sent via the internet or local network. In essence, IP addresses are the identifier that allows information to be sent between devices on a network: they contain location information and make devices accessible for communication.
